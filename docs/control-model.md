# Control Model

## Layered Security Model

The system applies a layered Zero Trust approach.

---

## Prevent

Controls intended to reduce attack likelihood.

- MFA enforcement
- Conditional Access
- Device compliance evaluation

---

## Detect

Controls intended to identify suspicious behavior.

- Sign-in risk analysis
- Behavioral anomaly detection
- SIEM alerts

---

## Contain

Controls intended to limit blast radius.

- Session revocation
- Account lock
- Device wipe

---

## Recover

Controls intended to restore access securely.

- Guided SSPR
- Helpdesk workflow
- Multi-step verification
